Billie Mars Time Microservice

Base URL: /, Version: 1.0.0

This microservice is used to convert the Earth UTC time into a Mars MSD and MTC equivalents.

Schemes: http

Summary

Path Operation Description
/mars-time/{earthUTCTime} GET

Converts the UTC time into a MSD and MTC equivalent

Ex: http://localhost/mars-time/convert/2020-07-11T16:36:52+00:00
